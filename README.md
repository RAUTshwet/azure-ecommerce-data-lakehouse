# azure-ecommerce-data-lakehouse


## Project Overview
This project demonstrates an end-to-end Azure Data Engineering solution using:

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- Azure Synapse Analytics
- Power BI

## Architecture

Source Data
     ↓
Azure Data Factory
     ↓
ADLS Gen2 (Bronze)
     ↓
Databricks (Silver)
     ↓
Delta Lake (Gold)
     ↓
Synapse Analytics
     ↓
Power BI Dashboard

## Features
- ETL Pipeline
- Incremental Loading
- Data Quality Checks
- Delta Lake Architecture
- Reporting Dashboard
